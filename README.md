# dmenu - dynamic menu
dmenu is an efficient dynamic menu for X.

## Main Source
This is a fork of dmenu from the suckless project:
http://git.suckless.org/dmenu

## Installation
Edit `config.mk` to match your local setup  
(dmenu is installed into the `/usr/local` namespace by default).

Afterwards, run the following command to build and install dmenu
(as root if necessary):

    make clean install
